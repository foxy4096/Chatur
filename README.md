## Setup and usage

Put this CDN link in HTML head

```
<link rel="stylesheet" href="https://gitcdn.link/repo/foxy4096/Chatur/main/chatur.css">
```
OR
```
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/foxy4096/Chatur/chatur.css">
```
